# Navigation

默认页面布局支持各种导航功能: 

* **主导航（main navigation）** 面板，突出显示当前选定的页面；
* 一个 **辅助（auxiliary）** 链接列表，显示在所有页面上；
* **breadcrumbs**列表,显示在除顶级页面之外的所有页面上；
* 指向**子页面（child pages）**（所谓的“目录”）的可选链接列表；以及
* 用于**页内导航（in-page navigation）**的部分标题链接，带有可选的**返回顶部（back-to-top）**链接。
